# Executive Summary

The Quote Builder is the **central hub** of R&G Consulting's proprietary toolset. It must be:
- Elegant and professional
- Intuitive at first glance
- Low friction for data input
- Responsive across ALL devices (mobile-first for restaurant industry)
- Infinitely smooth like Google Maps

This document outlines the complete UI/UX overhaul strategy.

# 6. Component Specifications

## 6.1 Infinite Canvas

### Behavior
- **Pan**: Two-finger drag (touch) or Space+drag (desktop) or middle-mouse
- **Zoom**: Pinch (touch) or Ctrl+scroll (desktop)
- **No boundaries**: Content can be placed anywhere; canvas extends infinitely
- **Performance**: Only render objects in viewport + buffer zone
- **Grid**: Scales with zoom, shows finer lines at higher zoom

### Technical Implementation
```javascript
// Canvas uses CSS transform for pan/zoom
// Objects positioned in "world coordinates"
// Viewport calculation determines what to render

const worldToScreen = (worldX, worldY, pan, zoom) => ({
  x: (worldX * zoom) + pan.x,
  y: (worldY * zoom) + pan.y
});

const screenToWorld = (screenX, screenY, pan, zoom) => ({
  x: (screenX - pan.x) / zoom,
  y: (screenY - pan.y) / zoom
});
```

### Grid Scaling
- Zoom < 50%: Show 10ft major lines only
- Zoom 50-100%: Show 10ft major, 1ft minor
- Zoom > 100%: Show 10ft major, 1ft minor, 0.5ft micro
- Zoom > 200%: Show inch marks

## 6.2 Fixed Rulers

### Position
- Horizontal ruler: Fixed to TOP of canvas viewport (below header)
- Vertical ruler: Fixed to LEFT of canvas viewport (after left panel)
- Both rulers show measurements relative to canvas origin

### Behavior
- Stay fixed while canvas pans
- Scale markers update based on zoom level
- Show current cursor position indicator

## 6.3 Collapsible Panels

### Animation
- Duration: 200ms
- Easing: ease-out
- Collapse: Width animates to 0 (desktop) or Height to 0 (mobile bottom sheet)
- Content fades out as panel collapses

### Touch Targets
- Minimum 44px Ã— 44px for all interactive elements
- Generous padding between items
- Clear visual feedback on tap/hover

- Cluttered appearance

- Overwhelming visual noise

- Flatten the hierarchy
- Use spacing/dividers instead of nested boxes
