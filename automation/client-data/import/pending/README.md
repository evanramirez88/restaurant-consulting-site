# Pending Import Folder

Drop files here to import into the client data system.

## Supported Formats
- `.txt` - Text files, notes
- `.md` - Markdown documents
- `.pdf` - PDF documents
- `.xlsx` - Excel spreadsheets
- `.csv` - CSV data files
- `.json` - JSON data
- `.xml` - XML exports (SMS, etc.)

Files will be processed and moved to `../processed/` or `../failed/`
